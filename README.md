Interfaces Incarnées et Augmentées
==================================

**Projet 2025-2026**

---

## Description du projet

- Prototype : détecter obstacles proches et fournir un retour multimodal compréhensible (vibration proportionnelle + son directionnel + repère visuel minimal si pertinent).
- Matériel minimal : capteur de distance (LiDAR/ToF), microcontrôleur/mini‑ordinateur, vibreur intégré (canne/bracelet), écouteurs ou petit haut‑parleur, option lunettes/smartphone pour indicateur visuel.
- Multimodal :
	- Haptique : vibration proportionnelle à la distance et patterns pour danger urgent.
	- Audio : son et tonalité pour priorité.

Usage type : Navigation urbaine pour personnes malvoyantes. Canne et lunettes pour détecter obstacles et danger environnant.
